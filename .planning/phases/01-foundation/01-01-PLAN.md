# Plan 01-01: Initialize Astro 5.17, Tailwind v4, and deployment config

## Goal

Scaffold an Astro 5.17 project with Tailwind CSS v4, typography plugin, remark reading-time plugin, and Netlify deployment config so that `npm run build` produces a deployable `dist/` directory.

## must_haves

- [ ] `npm run build` exits 0 and `dist/` directory exists
- [ ] `package.json` contains `"astro"` at version `5.17.0` and does NOT contain `@astrojs/tailwind`
- [ ] `astro.config.mjs` has `output: "static"`, `site: "https://dawoodkamar.com"`, `tailwindcss()` in vite plugins, `remarkReadingTime` in markdown plugins
- [ ] `netlify.toml` contains `command = "npm run build"` and `publish = "dist"`
- [ ] `src/styles/global.css` contains `@import "tailwindcss"`, `@plugin "@tailwindcss/typography"`, and `@custom-variant dark`
- [ ] `src/config.ts` exports `SITE_URL` and `SITE_TITLE`
- [ ] `src/plugins/remark-reading-time.mjs` exports `remarkReadingTime` function
- [ ] Built CSS in `dist/_astro/` contains the string `prose`

   ```typescript
   export const SITE_TITLE = "Dawood Kamar";
   ```

4. Create `src/plugins/remark-reading-time.mjs` with this exact content:
   ```javascript
   import getReadingTime from "reading-time";
   import { toString } from "mdast-util-to-string";

   export function remarkReadingTime() {
     return function (tree, { data }) {
       const textOnPage = toString(tree);
       const readingTime = getReadingTime(textOnPage);
       data.astro.frontmatter.minutesRead = readingTime.text;
     };
   }
   ```

5. Replace `src/pages/index.astro` (scaffold-generated) with this content that imports global CSS to wire Tailwind into the Vite module graph:

   ```astro
       <meta charset="utf-8" />
       <meta name="viewport" content="width=device-width, initial-scale=1" />
       <title>Dawood Kamar</title>
     </head>
     <body>
       <main class="prose mx-auto p-8">
         <h1 class="text-4xl font-bold">Dawood Kamar</h1>
         <p class="text-lg text-gray-600">Site under construction. Astro + Tailwind v4 foundation is live.</p>
       </main>
     </body>
   </html>
   ```
   This placeholder uses `prose`, `mx-auto`, `p-8`, `text-4xl`, `font-bold`, `text-lg`, and `text-gray-600` Tailwind classes to verify Tailwind v4 is generating CSS.
