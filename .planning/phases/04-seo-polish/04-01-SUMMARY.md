# Phase 4 Plan 1: Metadata, Structured Data, Sitemap, RSS, and robots.txt Summary

## One-liner

Full SEO head via BaseHead.astro (OG/Twitter/canonical/RSS autodiscovery), BlogPosting JSON-LD on blog posts via set:html, sitemap auto-generated by @astrojs/sitemap, RSS feed at /rss.xml with draft filter, and robots.txt with sitemap reference.

## What Was Built

### Task 1: BaseHead.astro, sitemap, robots.txt (commit 5327079)

Installed `@astrojs/sitemap` and `@astrojs/rss`. Created `src/components/BaseHead.astro` — a reusable head fragment component that emits: charset/viewport, canonical URL (computed via `new URL(Astro.url.pathname, Astro.site)`), title, description, Open Graph tags (type, url, title, description, image), Twitter Card tags, and RSS autodiscovery link. Created `public/robots.txt` with permissive crawl policy and Sitemap reference. Created `public/og-default.png` (minimal 1x1 PNG placeholder for og:image). Updated `astro.config.mjs` to register `sitemap()` integration. Updated `BaseLayout.astro` to replace inline meta tags with `<BaseHead />` and added a `<slot name="head" />` for per-page head injections.

**Validation passed:** `dist/sitemap-index.xml` generated, `og:title` present in `dist/index.html`, canonical link present, `dist/robots.txt` correct.

### Task 2: ArticleJsonLd, RSS feed, JSON-LD wired (commit 09f2933)

Created `src/components/ArticleJsonLd.astro` — emits a `<script type="application/ld+json">` using `set:html={JSON.stringify(schema)}` to prevent Astro from HTML-escaping double-quotes. The schema is `BlogPosting` with headline, description, datePublished, url, and author. Created `src/pages/rss.xml.js` as a GET endpoint using `@astrojs/rss`, filtering drafts with `data.draft !== true`, using `context.site` for the site URL. Updated `src/pages/blog/[id].astro` to import `ArticleJsonLd` and render it via `slot="head"` inside BaseLayout.

**Validation passed:** `dist/rss.xml` present with 2 published posts, JSON-LD blocks present on both post pages with unescaped `"` quotes, OG tags on all 6 pages.

## Deviations from Plan

None — plan executed exactly as written.
